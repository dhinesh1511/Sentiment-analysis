# Sentiment-analysis
I have used 6 models:
* Logistic Regression
* Support Vector Classifier
* Random Forest Classifier
* LSTM
* Bidirectional LSTM
* Decision Tree

## Accuracy scores of the models:
Logistic Regression -      64%  
Support Vector Classifier - 63%   
Random Forest Classifier -  62%   
LSTM -                    64.5%   
Bidirectional LSTM -      64.3%  
Decision tree             -60%
